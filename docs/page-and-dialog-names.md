# 페이지·팝업 명칭 정리

앱 코드 기준(헤더 `DialogTitle` / `AlertDialogTitle` / `CardTitle` 등)으로 정리했습니다. 동적 문자열은 패턴으로 표기합니다.

---

## 페이지(화면)

| 경로(URL) | 화면 명칭(헤더·카드 등) | 메타/비고 |
|-----------|-------------------------|-----------|
| `/` | 포장 예약 | `app/layout.tsx` 문서 제목도 동일 |
| `/rest-menu` | 식당 메뉴 | |
| `/login` | 로그인 | 카드 제목 |
| `/signup` | 회원가입 | 카드 제목 |

## 팝업·다이얼로그 — 포장 예약(`/`, `home-page-client.tsx`)

| 구분 | 제목(명칭) | 비고 |
|------|------------|------|
| 식당 상세 | *(선택한 식당 이름)* | `DialogTitle` |
| 제로페이 선결제 확인 | 제로페이 선결제 | 선결제 import 확인용 `Dialog` |
| 메뉴 히스토리 | 메뉴 히스토리 | |
| 등록 메뉴 목록 | 등록된 메뉴 / *{식당명}* 메뉴 | 식당명 유무에 따라 |
| 종류 선택 | 종류 선택 | |
| 식당 등록·수정 | 식당 등록 / *{식당 ID}* | 수정 시 제목이 ID |
| 테마 설정 | 테마 설정 | `AlertDialog` |
| 삭제 확인 | 삭제하시겠어요? | 예약·선결제 삭제 확인 |
| 공유 미리보기 | 공유 양식 미리보기 | |

## 팝업·다이얼로그 — 식당 메뉴(`/rest-menu`, `rest-menu-client.tsx`)

| 구분 | 제목(명칭) | 비고 |
|------|------------|------|
| 식당 메뉴 보기 | *{식당명}* / 식당 메뉴 | 식당 미선택 시 fallback |
| 알림에서 메뉴 가져오기 | 메뉴 등록 | 결제 알림 큐가 있을 때, 메뉴명 입력 후 확인 |
| 메뉴 삭제 확인 | 메뉴 삭제 | `AlertDialog` |
| 방문 메뉴 이력 | *{식당명}* | 선택 식당 이름만 제목으로 사용 |
| 테마 설정 | 테마 설정 | `AlertDialog` |
| 이미지 보기 | *(제목 없음, 이미지 영역)* | 대체 텍스트: 메뉴 이미지 |

## 드롭다운 등(페이지 헤더, 명칭만)

포장 예약 상단 메뉴 항목 예: 식당 메뉴, 식당 등록, 테마, 식당 종류.  
식당 메뉴 상단 메뉴 항목 예: 포장 예약, 식당 등록, 테마, 식당 종류.

(별도 `DialogTitle`은 아니며, 메뉴 라벨로 표시됩니다.)
